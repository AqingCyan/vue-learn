# 如何根据该份指南学习 Vue 呢？

  1. 手敲每一个 demo，先阅读 demo，然后再阅读笔记，之后再根据笔记写出 demo。

  2. 紧跟Vue 官方文档学习，文档中涵盖的内容比笔记全面，笔记只是更加通俗易懂的解释且是 demo 理解的辅助内容。 对于你我这样的学习者，通俗的话往往更加容易理解其中深刻的道理。

  3. 如果仍觉的 demo 与笔记晦涩难懂，可以在网上寻找网课学习，这里推荐我当时学习观看的网课。

  4. 在敲第一个 demo 之前，首先确保你已经准备好切换开发思维，采用一种全新的开发思维去实现我们的学习成长。

TIP

网课请支持正版，尊重他人劳动成果。😄
